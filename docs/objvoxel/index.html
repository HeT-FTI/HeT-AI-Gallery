鸡翅智能测重系统

产品简介

鸡翅智能测量系统是智能烤箱的核心功能模块，专为家庭厨房场景设计。 通过内置摄像头拍摄图像，系统可自动完成鸡翅的尺寸测量和重量估算。

核心优势：

  • 非接触式测量，无需额外操作，放入即可识别

  • 快速响应，测量结果即时反馈给烤箱控制系统

  • 重量估算误差 < %，满足家庭烹饪精度要求

功能特性

透视畸变校正

当相机以倾斜角度拍摄时，图像会产生透视畸变——近处物体显得更大，远处物体显得更小。 系统自动校正这种畸变，确保测量结果准确可靠。

尺寸自动测量

系统自动识别鸡翅轮廓，提取最小外接矩形，获取长轴和短轴尺寸， 为后续体积估算和重量计算提供基础数据。

智能重量估算

采用椭球体体积模型，根据测量尺寸自动计算鸡翅体积，并结合鸡肉密度参数估算重量。 无需逐个称重，即可获得可靠的重量数据。

数据输出接口

系统为嵌入式平台提供简洁的数据输出接口，测量完成后直接输出结构化数据， 便于智能烤箱主控系统调用：
