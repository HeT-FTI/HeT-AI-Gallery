IE Project Documentation

Introduction

IE (Inference Engine) is a deep learning inference framework designed specifically for the embedded field based on the ANSC C99 standard.

Main Features

  • Rapid development: The trained model (currently only supports .flite) can be directly converted into a code file or .bin file through tools, and quickly deployed in the project

  • Supports complex structures: Supports multiple input and multi branch network structures

  • Lightweight: Adopting time-division multiplexing for data structure design to ensure minimal resource consumption and facilitate integration into MCU

  • Single dependency: pure C development, minimum support for C99 standard, relying only on a single SMT library, no other third-party library dependencies, easy cross platform porting

  • High performance: can effectively utilize the acceleration capabilities of hardware platforms such as multi-core computing and SIMD parallelism

  • Trimmable: Supports operators to freely crop according to actual applications

  • Multi backend architecture: Reserved CPU/GPU/NPU backend interfaces, currently implemented CPU backend, supports custom memory allocation and operator registration

Quick Start

Code 1 Using IE reasoning classification tasks
#include "ie_api.h"

int main(void)
{
   /* 1. Prepare to input data */
   IeTensorInfo inputs = {
      .dType = IE_DTYPE_UT8,
      .shape = {1, 224, 224, 3},
      .pData = image_data,
   };

   /* 2. Load model binary data and initialize the calculation graph */
   IeGraph graph = {0};
   const void * pAttrBlob = load_bin("./ie_gAttrBlob.bin");
   const void * pShapeBlob = load_bin("./ie_gShapeBlob.bin");
   const void * pTensorDesc = load_bin("./ie_gTensorDesc.bin");
   const void * pLayerDesc = load_bin("./ie_gLayerDesc.bin");
   const void * pDataBlob = load_bin("./ie_gDataBlob.bin");
   ie_graph_init(&graph, pLayerDesc, pTensorDesc, NULL, pDataBlob, pAttrBlob, pShapeBlob,
                  ie_backend_init(NULL, NULL, NULL, 0), tensors_count, layers_count);

   /* 3. Set input */
   ie_graph_set_input(&graph, &inputs);

   /* 4. Execute inference */
   ie_graph_compute(&graph);

   /* 5. Get output and post-process */
   IeClassification classification = {0};
   ie_classification(ie_graph_get_output(&graph, out_tensor_index), &classification);

   /* 6. Release resources */
   ie_graph_release(&graph);

   return 0;
}

Project Structure

IE/
├── include/                   # 公共头文件
│   ├── ie_api.h               # 统一 API 入口
│   ├── ie_def.h               # 基础类型和宏定义
│   ├── api/                   # 对外 API 模块头文件
│   │   ├── ie_api_graph.h     # 计算图 API
│   │   ├── ie_api_backend.h   # 后端 API
│   │   └── ie_api_postprocess.h  # 后处理 API
│   └── private/               # 内部头文件
│       ├── ie_struct.h        # 内部数据结构
│       ├── ie_operator.h      # 算子定义
│       └── config/            # 可配置选项
│           ├── ie_config_basic.h # 全局配置
│           ├── ie_config_op.h # 算子启用配置
│           └── ie_port.h      # 平台移植配置
├── src/                       # 源代码实现
│   ├── ie_graph.c             # 计算图核心逻辑
│   ├── ie_tensor.c            # 张量操作
│   ├── backend/               # 后端实现
│   │   ├── ie_backend.c       # 后端接口
│   │   └── cpu/               # CPU 后端
|   │   └── gpu/               # GPU 后端
|   │   └── npu/               # NPU 后端
│   ├── layers/                # 网络层算子实现
│   └── postprocess/           # 后处理模块
├── SMT/                       # 内置数学库子模块
├── examples/                  # 示例应用
├── test_package/              # 测试工程
│   └── test/
│       ├── unit/              # 单元测试
│       └── stress/            # 压力测试
├── docs/                      # 文档系统
├── CMakeLists.txt             # CMake 构建配置
├── conanfile.py               # Conan 包管理配置
└── metadata.json              # 项目元数据
